🧑‍🍳 사장님(업주) 전용 인터페이스 
메뉴 구조 제안
1. 대시보드

오늘/주간/월간 노출수·클릭수·전환율·매출추정치 확인 (쿠폰 관련 지표 제외).

daily_merchant_kpis 집계 데이터 사용 (impression, click, benefit_view 등).

기간·매장·카테고리별 필터 가능.

주요 지표:

노출수(Impressions)

클릭수(Clicks)

전환율(Click → 상세 진입)

매출추정치(혜택 사용 추정가 기반)

2. 매장 관리

merchants 테이블과 연동:

매장 기본정보 수정 (주소, 전화번호, 카테고리, 영업시간).

휴무일·예외시간 등록(merchant_hours, merchant_hour_exceptions).

대표 이미지 1장 업로드·수정·삭제(merchant_images).

지도 기반 좌표 선택 및 위치 표시.

3. 혜택 관리

benefits 테이블과 연동.

게시 중(PUBLISHED)/일시정지(PAUSED)/심사중 상태별 목록 표시.

새 혜택 등록 및 수정:

유형 선택: 정액 할인(AMOUNT) / 정률 할인(PERCENT) / 증정(GIFT) / 멤버십(MEMBERSHIP)

유효기간, 블랙아웃, 시간대, 적용 지역 설정.

지오펜싱 반경 설정(geo_radius_m).

발급 한도·쿠폰 로직은 제거됨.

상태 변경:

일시정지/재게시 기능만 유지.

4. 성과 분석

event_logs 및 daily_merchant_kpis 기반으로 노출·클릭·전환률 중심의 그래프 제공.

혜택별·시간대별·카테고리별 상세 분석.

지도 히트맵, 권역별 분포, 인기 혜택 TOP5 표시.

5. 직원 권한 관리

user_roles 관리화면에서 매장별 계정 초대 및 권한 부여/철회.

직원은 본인 매장 범위에서만 혜택·매장 데이터 접근 가능.

초대는 이메일 기반 토큰 또는 링크 생성 방식.

🛠 관리자 전용 인터페이스 (쿠폰 기능 제거 버전)
메뉴 구조 제안
1. 종합 대시보드

전체 가맹점/혜택 운영 현황 및 트래픽 지표 요약.

혜택 등록 수, 게시 상태별 비율, 카테고리별 노출/클릭 비중, 신고 통계 등.

2. 가맹점 관리

merchants 테이블 CRUD.

매장 정보 수정, 카테고리 변경, 이미지 수정(merchant_images).

영업시간·휴무일 관리.

신규 등록 요청 승인/반려/삭제/일시정지.

중복 매장 병합 처리.

지도 위치 수정 가능.

3. 혜택 관리

benefits 테이블 연동 (쿠폰 테이블 제거됨).

상태별 필터링 및 검색.

워크플로(DRAFT → SUBMITTED → APPROVED → PUBLISHED → PAUSED/EXPIRED → REJECTED).

심사·승인·반려·버전 롤백 가능.

CSV 일괄 업로드 기능 유지.

쿠폰 데이터(coupons, coupon_redemptions) 관련 컬럼 및 메뉴 제거.

4. 카테고리/권역 관리

categories, regions 테이블 CRUD.

카테고리명·순서·아이콘(icon_url)·색상(marker_color) 수정.

권역(시·구·동, 커스텀 경계) 추가/수정.

5. 배너/게시물 관리

home_banners 테이블 연동.

제목, 부제목, 이미지, 링크, 순서, 노출여부 관리.

이미지 업로드/삭제 가능.

게시물 프리뷰 기능.

6. 계정 관리 (사용자/사장님)

users, roles, user_roles 테이블 기반.

계정 목록/검색/필터.

상태 변경(활성화/일시정지/삭제).

가입 승인·거절.

사장님 신청 계정 승인 프로세스 포함.

각 계정의 status 컬럼 업데이트로 실시간 반영.

7. 권한 및 감사 로그

permissions, role_permissions 관리.

역할별 접근 권한 수정.

admin_audit_logs를 통해 변경 이력 열람·다운로드.
